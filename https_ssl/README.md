# HTTPS SSL

This project configures SSL/TLS encryption and HTTPS for web servers using HAProxy.

## Tasks

### 0. World wide web
Script to display DNS information about subdomains.

### 1. HAproxy SSL termination
Configure HAProxy to handle SSL/TLS encryption on port 443.

### 2. No loophole in your website traffic
Redirect all HTTP traffic to HTTPS automatically.
